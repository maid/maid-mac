# maid-mac

macOS GUI for Maid, written in MacRuby.

**Maintainer wanted!** Please get in touch if you're interested.

## Installation

To install Maid Mac, open up Terminal and `cd` the path of the Maid app.

```
$ cd ~/Downloads/maid-mac
$ macruby app.rb
```
